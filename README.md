# Prompt-Based Stance Control in German: An Evaluation of LLMs for Experimental Research on Attitude Change
This repository contains all the scripts used in the paper *Prompt-Based Stance Control in German: An Evaluation of LLMs for Experimental Research on Attitude Change*, published at the `<TODO>`.

# Usage
All scripts used to carry out the selection of stance detection method, are located in the `Stance Detection` folder.
All scripts used to carry out the experiments on LLM-based stance control are located in the `Stance Control` folder.
For both there is a seperat readme.md in the respective folder containing more detailed information on the code and experiment reproduction.

The experiments on [stance detection](https://github.com/Prompt-based-Stance-Control-for-LLMs/Prompt-based_Stance_Control_for_LLMs/tree/main/Experiment1) comprise
 - training of a BERT-based classifier
 - LLM-based classification
 - aggregation/evaluation of results

The experiments on  [stance control](https://github.com/Prompt-based-Stance-Control-for-LLMs/Prompt-based_Stance_Control_for_LLMs/tree/main/Experiment2) methods comprise
 - Generating responses with (controlled) LLMs
 - Detecting the stance of the responses
 - Aggregation/Evaluation of results
